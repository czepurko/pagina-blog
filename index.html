Surgimento da Internet

(Voltar)

A internet surgiu a partir de um projeto militar dos Estados Unidos, chamado ARPANET, criado em 1969 pela agência DARPA. O objetivo era permitir a comunicação entre computadores mesmo que uma parte da rede fosse destruída, por conta do contexto da Guerra Fria. No começo, conectava apenas universidades e centros de pesquisa.

Com o tempo, especialmente nos anos 1980, a rede foi crescendo, e surgiram protocolos fundamentais como o TCP/IP, que permitiram que diferentes redes se comunicassem entre si — isso foi o embrião da internet como conhecemos hoje.

Já nos anos 1990, com a criação da World Wide Web por Tim Berners-Lee, a internet deu um salto: as páginas passaram a ter links, imagens e se tornaram acessíveis ao público. Isso popularizou o acesso e mudou para sempre a forma como o mundo se comunica, estuda e trabalha.

Desde então, a internet evoluiu rápido, mas sempre com base nesses princípios lá de trás: descentralização, conexão e compartilhamento de informações.

Como funciona o HTML

(Voltar)

O HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas da web. Ele estrutura o conteúdo, dizendo ao navegador o que é um título, um parágrafo, uma imagem, um link, uma lista, etc. Funciona com tags (ou elementos), que são marcadores envoltos por<>. Exemplo: h1 para título principal, p para parágrafo, img para imagem. Quase toda tag tem uma tag de abertura (p) e uma de fechamento (p), mas algumas são autofechadas, como img.

HTML não é uma linguagem de programação, e sim de marcação — ele não faz lógica, só organiza o conteúdo. O navegador lê o HTML de cima pra baixo, renderizando a página conforme encontra os elementos. O HTML pode ter atributos, que dão mais informações às tags. Por exemplo, em img src="foto.jpg" alt="Minha Foto", src indica o caminho da imagem.

Ele trabalha junto com CSS (estiliza) e JavaScript (dá interatividade), formando o trio principal do front-end. Por fim, mesmo com novas tecnologias, o HTML continua sendo a base da web, e entender ele bem é essencial pra todo programador.

O que são Tags

(Voltar)

Tags são os blocos básicos do HTML. Elas servem para marcar e organizar o conteúdo de uma página. É com elas que o navegador entende o que mostrar e como mostrar. Cada tag tem uma função específica, por exemplo: h1 até h6 → títulos, do maior pro menor. p → parágrafos de texto. a → links. img → imagens. ul e li → listas. As tags geralmente aparecem assim: Abertura: p Conteúdo: Olá, mundo! Fechamento: p Juntas ficam: Olá, mundo! Algumas são autofechadas, tipo imagens: img src="foto.jpg" Pensa nas tags como os tijolos que constroem um site. Cada tag tem seu papel, e o navegador lê isso pra montar a página. Ou seja, sem tag, sem site. Elas são a alma do HTML!
